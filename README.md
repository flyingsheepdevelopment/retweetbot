Requirements
------------

 * Python
 * OAuth2

Setup
-----

 * Put bot.py, log.py, my_bot.py and _twitter.py in the same directory
 * Edit the config and API sections in my_bot.py
 * You can rename my_bot.py

Run
---

 * Execute: python my_bot.py
